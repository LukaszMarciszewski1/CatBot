## Run Locally

### 1. Clone repo

```
You can just clone repository files and run to install all dependencies:

Node (version 14.17.0)

```
### 2. Install Backend

$ npm install

### 3. Install Frontend

```
# open new terminal
$ cd client
$ npm install

```

### 4. Run app

$ cd .. 
$ npm run dev
